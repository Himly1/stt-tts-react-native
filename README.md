# stt-tts-react-native

A react-native library of STT and TTS with tensorflow

## Installation

```sh
npm install stt-tts-react-native
```

## Usage

```js
import { multiply } from 'stt-tts-react-native';

// ...

const result = await multiply(3, 7);
```
